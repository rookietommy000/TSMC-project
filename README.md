## 功能簡介

此專案為職訓課程練習，獨立作業練習出產品上架功能

- 無框架
- 在後台管理系統可以管理商品的類別和其上架，以及會員的管理，還有消息的新增
- 前台網頁可以進行會員登入、查看商品上架、消息等渲染效果
